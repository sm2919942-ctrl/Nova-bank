#  Nova Bank - Full Stack Banking Application

Nova Bank is a secure and modern full-stack web application designed to simulate real-world digital banking operations. It integrates a robust Spring Boot backend with a responsive client interface to handle core financial services efficiently.

---

##  Key Features

* **User Authentication:** Secure user signup and login managed via Spring Security and encrypted with BCrypt.
* **Account Management:** Real-time balance checks, profile tracking, and unique account identifier assignments.
* **Transaction Services:** Core modules to handle funds deposition, withdrawals, and inter-account transfers securely.
* **Responsive Architecture:** A modern frontend layout built to adjust across multiple screen displays.

---

##  Tech Stack

### Backend
* **Framework:** Spring Boot (Java)
* **Security:** Spring Security & BCrypt Encryption
* **Database / Architecture:** RESTful APIs, Entity Relational Design

### Frontend
* **Core Languages:** HTML5, CSS3, JavaScript (Modern Web Interactivity)
